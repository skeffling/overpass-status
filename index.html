📋 Welcome to Overpass API Status Monitor

This tool helps you monitor the health and performance of Overpass API servers — the query interface for OpenStreetMap data. Whether you're a developer building mapping applications or a researcher analyzing geographic data, this dashboard gives you real-time visibility into server availability. Use it to monitor public Overpass servers like overpass-api.de, or add your own self-hosted instances to keep an eye on your infrastructure.

What we monitor for each server:

  • Latency — Response time to the status endpoint
  • Query Slots — Available capacity for new queries (rate limiting)
  • Active Queries — Currently running queries on the server
  • Data Age — How recently the server's OSM data was updated
  • Version — Overpass API software version and git commit

Color coding at a glance:

  • ● Green — Healthy (fast latency, slots available, fresh data, latest version)
  • ● Orange — Warning (moderate latency, some slots in use, slightly stale data, older version)
  • ● Red — Issue (slow latency, no slots available, stale data)

Add servers using the button below, or expand Landmark Tests to run queries against iconic places like the Eiffel Tower, Statue of Liberty, and Sydney Opera House.

About Overpass Status

Overpass Status is a browser-based monitoring tool for Overpass API servers.

Features

  • Server Monitoring - Check status, latency, available slots, and active queries
  • Data Freshness - See how recently each server's OSM data was updated
  • Landmark Tests - Verify servers are returning real data by querying famous locations worldwide
  • Multi-Server - Compare multiple public or self-hosted Overpass instances
  • API Logging - View all requests and responses in real-time

How It Works

This tool runs entirely in your browser - no backend server required. It makes direct API calls to Overpass servers and displays the results. Your server list and preferences are saved locally in your browser.
